## Keybinds
When running the app, instead of the buttons you can use the keybinds described below.

| Keybind  | Descrition            |
|----------|-----------------------|
| Ctrl + S | Save image locally.   |
| Ctrl + N | Fetch the next image. |
| Ctrl + O | Open options.         |

> [!NOTE]
> Options have not yet been implemented.
